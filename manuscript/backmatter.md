# コンパニオンキットについて

本書は、無償の解説レイヤーである。有償の実行レイヤーは別製品、すなわち **AIエージェントセキュリティ検証キット** である。

本書は*何を*検証すべきか、そして*なぜ*かを教える。キットは検証を*実行*し、証跡を生成する。

- 1コマンドで実行でき、対象のエージェント設定に対して PASS/FAIL を返す、実行可能なインジェクションテストスイートと MCP/EMA 設定チェックリスト
- 機械可読な JSON 結果と人間可読な PDF レポートを1つにまとめた、単一のビルド成果物
- その成果物に対する RFC 3161 タイムスタンプ。これにより、結果は CISO や監査人がそのまま受け入れられる、改竄検知可能な監査記録になる

このキットが存在するのは、「我々はエージェントをテストした」という言葉が証跡なしには何の価値もなく、その証跡を手作業で作ることこそが退屈な部分だからである。詳細、エディション（Solo / Team）、更新サブスクリプションについては、変更履歴に記載の配布ページを参照。
